Migration workflow

Po ověření UI Kitu začni přepisovat sekce Genetia v1 → v2 po částech (section-by-section) a používej pouze nové role + komponenty.

1) Text roles

Používej page/section/card role napříč webem. Výjimky řeš přes CSS proměnné.

2) Components

Buttons, cards, badges, modal, tabs, accordion, forms… vše pouze z assets/css/components.

3) Page layouts

Stránkové CSS patří jen do assets/css/pages. Bez duplicit a bez globálních přepisů.
